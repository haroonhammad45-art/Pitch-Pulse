# PitchPulse Athlete Wellness

PitchPulse is a cricket athlete wellness app prototype created for tracking:

- Daily mental wellness check-ins
- Burnout and fatigue risk
- Bowling workload
- Gym workouts
- Food calories and protein
- Sleep, recovery, hydration, and reminders
- Fast bowler injury awareness
- Cricket psychology routines

## How to Open

Open `index.html` in any browser.

## How to Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload everything inside this folder.
3. Go to repository `Settings`.
4. Open `Pages`.
5. Choose `Deploy from a branch`.
6. Select the `main` branch and `/root`.
7. Save.

GitHub will give you a public website link after it finishes publishing.
